# 🏨 กลุ่มที่ 2: ระบบแสดงรายการสินค้าและหมวดหมู่ (หมวดอุปกรณ์โรงแรม)

โฟลเดอร์นี้บรรจุโค้ดและทรัพยากรสำหรับแอปพลิเคชัน `products` ซึ่งรับผิดชอบระบบจัดการสินค้าอุปกรณ์โรงแรมและของใช้พรีเมียม

## 📋 รายละเอียดความต้องการ (Requirements)

| Requirement ID | รายละเอียดความต้องการ (Description) |
| :--- | :--- |
| **FR-05** | ระบบต้องสามารถแสดงรายการสินค้าทั้งหมดในรูปแบบ Grid ที่สวยงาม |
| **FR-06** | ระบบต้องสามารถแยกประเภทและกรองสินค้าตามหมวดหมู่ได้ |
| **FR-07** | ระบบต้องสามารถแสดงข้อมูลรายละเอียดสินค้าเชิงลึก (ราคา, สต็อก, คำอธิบาย) |
| **FR-08** | ระบบต้องมีช่องสำหรับค้นหาสินค้าอัจฉริยะ (Search Engine) |

## 📂 โครงสร้างไฟล์สำคัญ
- `models.py`: จัดเก็บโครงสร้างข้อมูล Category และ Product (รองรับภาษาไทยใน URL)
- `views.py`: ควบคุมตรรกะการแสดงผล การกรอง และการค้นหา
- `urls.py`: กำหนดเส้นทาง URL สำหรับเข้าถึงหน้าต่างๆ
- `templates/products/`: หน้าจอ UI สำหรับฝั่งผู้ใช้งานและผู้ดูแลระบบ
- `management/commands/seed_hotel_data.py`: สคริปต์สำหรับสร้างข้อมูลตัวอย่างอุปกรณ์โรงแรม 10 หมวดหมู่
